Telegram HTML to TXT Converter

Конвертер экспорта чатов Telegram из HTML в TXT формат


Описание

Данное приложение предназначено для конвертации экспортированных чатов Telegram из формата HTML в удобочитаемый текстовый формат. Программа автоматически обрабатывает все файлы экспорта (messages.html, messages2.html и т.д.) и объединяет их в один общий TXT-файл или DOCX.

Автор: sshteo


Возможности

- Автоматическое обнаружение всех HTML-файлов экспорта в папке
- Читаемый формат вывода: дата / ник и сообщение с ;
- Поддержка UTF-8 для корректного отображения русского языка
- Объединение нескольких файлов экспорта в один TXT-файл или DOCX
- Обработка медиафайлов (стикеры, фото помечаются специальными тегами)
- Полностью консольное приложение, не требует графического интерфейса
- Сохранение в форматах: TXT, DOCX, JSON, CSV
- Статистика чата (количество сообщений, активность участников)
- Фильтрация сообщений по дате


Быстрый старт

Системные требования

- .NET 8.0 SDK или выше
- Visual Studio Code (рекомендуется) или любая среда разработки для C#

Установка

Клонируйте репозиторий:
git clone https://github.com/sshteo/TelegramChatConverter.git
cd telegram-chat-converter

Восстановите зависимости:
dotnet restore

Запуск

dotnet run

При запуске программа попросит указать путь к папке с экспортом. Можно:

- Ввести полный путь к папке
- Нажать Enter, чтобы использовать текущую папку


Формат вывода

Каждое сообщение записывается в следующем формате:

дата / ник
текст сообщения;

Пример:

23.06.2023 13:34 / user1
Привет;

23.06.2023 13:34 / user2
здравствуй;

Конфигурация

Программа не требует дополнительной настройки. Все параметры передаются через аргументы командной строки.

Аргументы командной строки:
dotnet run [путь_к_папке]

путь_к_папке - необязательный параметр, путь к папке с экспортом


Пример использования

1. Экспортируйте чат из Telegram в формате HTML
2. Поместите все файлы (messages.html, messages2.html и т.д.) в одну папку
3. Запустите программу и укажите путь к этой папке
4. В папке с экспортом появится файл chat_export.txt


Меню программы

После обработки всех HTML-файлов программа предлагает выбор действий:

1. Сохранить в TXT (с разбивкой на части по 50 000 сообщений)
2. Сохранить в DOCX (один файл, открывается в Word)
3. Сохранить в JSON (структурированные данные)
4. Сохранить в CSV (таблица с разделением по столбцам)
5. Показать статистику чата
6. Экспорт по дате (фильтрация)
7. ВСЁ СРАЗУ (все форматы + статистика)
0. Выход


Статистика чата

При выборе пункта 5 программа показывает:

- Общее количество сообщений
- Период чата (первая и последняя дата)
- Самые активные дни (топ-10)
- Активность участников (количество и процент сообщений)
- Типы сообщений (текстовые и медиа)
- Средняя длина сообщения
- Самое длинное сообщение


Экспорт по дате

При выборе пункта 6 программа:

1. Запрашивает дату в формате ДД.ММ.ГГГГ
2. Находит все сообщения за указанную дату
3. Сохраняет их в отдельный TXT-файл

Известные проблемы

- Медиафайлы (фото, видео, стикеры) не сохраняются в TXT-файл, вместо них вставляются специальные метки
- При очень больших чатах (более 500 000 сообщений) обработка может занять длительное время
